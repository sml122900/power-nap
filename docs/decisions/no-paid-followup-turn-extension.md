# 후속 질문 유료 연장 대신 무료 상한 상향 — 실측 원가 기반 결정

## Problem

AI 분석 리포트의 후속 질문은 기본 3턴이 소진되면 입력창이 비활성됐다. 다음
기능으로 "이용권 1개(1,000원)로 질문 3회 더" 같은 유료 연장 상품을 만드는
방안이 제시됐다 — 기존 인앱상품(추가 분석 1,000원)과 같은 가격대로, 새 상품을
Play Console에 등록하는 안이었다.

## Action

상품을 설계하기 전에 실제 원가부터 확인했다. `analyses` 테이블의
`tokens_in`/`tokens_out`(리포트+후속 턴 누적)을 5건 조회해 Claude Sonnet 5 단가
($2/$10 per MTok, 도입가 기준)로 환산한 결과: 리포트 1회 평균 ≈ 53원, 3턴
풀소진 ≈ 64원. 후속 질문 콜은 `records_snapshot`(리포트 원가의 대부분을 차지하는
낮잠 기록 배열)을 재전송하지 않는 구조라, 후속 턴 자체의 한계 원가는 턴당
20원 안팎으로 극히 낮다는 걸 확인했다. 12턴(기본 3 + 구매 3회)까지 확장하는
최악의 시나리오까지 컨텍스트 누적을 감안해 추정해도 총 원가는 100원대에
그쳤다.

## Result

1,000원짜리 상품을 20원짜리 원가로 파는 셈이라 판단해 유료 연장 상품 자체를
만들지 않기로 했다 — 가격 대비 과도해 판매가 안 될 가능성이 크고, 상품이 하나
늘면 Play Console 등록·심사 부담이 배가된다는 점도 반영했다. 대신 무료 턴
상한을 3 → 10으로 올렸다(비용이 아니라 "컨텍스트 누적에 따른 답변 품질 저하"가
실제 상한 근거임을 리스크 메모에 명시). 새 인앱상품 없이 기존 가격 구조를
그대로 유지하면서 사용자 가치만 늘린 결정 — 원가를 먼저 측정하지 않았다면
불필요한 상품과 등록 부담을 만들었을 상황이었다.
