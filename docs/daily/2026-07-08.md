## 이슈

- config plugin의 문자열 매칭 패치가 처음엔 아무 에러 없이 그냥 실패했다 —
  `node_modules` 안 라이브러리 소스가 CRLF였는데 패치 코드의 비교 문자열은 LF였던 것.
  줄 단위로 스플라이스하는 방식으로 바꿔 개행 문자에 무관하게 만듦(상세는 트러블슈팅
  문서 참고)
- Android `windowSoftInputMode=adjustResize`만으로는 키보드가 떠도 스크롤 위치가
  따라가지 않아 커피냅 직접입력 폼이 계속 가려짐 — `KeyboardAvoidingView`를
  `behavior: 'height'`로 명시하고 포커스 시 지연 스크롤을 추가해서야 해결
- 이번 세션에서 처음으로 여러 브랜치를 동시에 다루면서 git worktree를 실전 활용 —
