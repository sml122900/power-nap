## 이슈

- keytool을 Claude Code 세션 안(`!` 프리픽스)에서 실행했더니 비밀번호 프롬프트가
  입력을 못 받아 3회 연속 빈 문자열로 처리되며 잠김 — harness가 중계하는
  콘솔이 마스킹 입력에 필요한 실제 TTY를 못 주는 것으로 판단, 완전히 별도인
  터미널 창에서 실행하도록 전환해 해결.
- `gradlew assembleRelease`가 `JAVA_HOME`이 없어 실패 — Android Studio 번들
  JBR(`C:\Program Files\Android\Android Studio\jbr`)로 우회. 이 환경변수는
  세션 로컬이라 다음 세션에서도 다시 잡아야 함(영구화하려면 시스템 환경변수
  등록 필요).
